En esta Práctica 1 construirás la estructura básica de una página HTML, incluyendo título, encabezados, párrafos y formato de texto (negritas, cursivas) como primer acercamiento a la codificación.

📌 Categoría central: Fundamentos de HTML
🎨 Saber: identificar etiquetas estructurales
🛠️ Saber hacer: escribir un documento HTML válido

📋 1. Objetivo de la práctica

Crear un archivo HTML con la estructura básica (<html>, <head>, <body>) que muestre:

📌 Un título en pestaña (title) 🙌 Letrero de BIENVENIDA ✏️ Párrafos con negritas (b) y cursivas (i) 👤 Nombre del alumno con etiqueta <b>

Nombre del archivo: miprimerapaginaweb.html (guardar en una carpeta independiente, por ejemplo: practica1).

📝 miprimerapaginaweb.html

<!DOCTYPE html> <html> <head> <title>Colegio de Bachilleres de Estado de Mexico</title> </head> <body> <p> <b>¡BIENVENIDO!</b></p> <p> Esta es la página del plantel <b>Escribe el número de tu plantel</b> </p> <p>Aquí recibirás los conocimientos esenciales para tu vida <i>laboral</i> o <i>universitaria</i></p> <p><b>Elaborado por:</b> Escribe tu nombre completo</p> </body> </html>

🔁 Sustituye "Escribe el número de tu plantel" por tu número de plantel (ej. 03) y coloca tu nombre completo donde se indica.

🧐 Etiquetas utilizadas (lo que debes identificar)

<html> </html> → raíz del documento
<head> → metadatos, título
<title> → título de pestaña
<body> → contenido visible
<p> → párrafo
<b> → negrita
<i> → cursiva

✅ Requerimientos para evaluación (Práctica 1)

🔹 1. Nombre en pestaña: "Colegio de Bachilleres del Estado de Mexico"
🔹 2. Letrero "¡BIENVENIDO!" en negritas
🔹 3. Párrafo con número de plantel en negritas
🔹 4. Palabras "laboral" y "universitaria" en cursiva
🔹 5. Nombre del autor precedido de "Elaborado por:" en negritas
🔹 6. Estructura básica completa: <html>, <head>, <body>
🔹 7. Archivo guardado como miprimerapaginaweb.html
🔹 8. Visualización correcta en navegador

📊 Ponderación: 7.5% de la calificación total (según encuadre).

🧠 Tips para tu primera página

  • 📁 Crea una carpeta llamada practica1 y guarda dentro el archivo miprimerapaginaweb.html.
  • 🔤 No uses caracteres especiales (ñ, acentos, espacios) en el nombre del archivo.
  • 🌐 Abre el archivo con Chrome, Firefox o Edge para ver el resultado.
  • ✏️ Si no ves los cambios, guarda de nuevo y recarga el navegador (Ctrl+F5).
  • 💡 Prueba modificando el texto o agregando más párrafos para experimentar.
